# Умное портфолио с фильтрацией

## Описание проекта
Интерактивное портфолио проектов с динамической фильтрацией по технологиям и сложности. Фильтры сохраняются в LocalStorage браузера.

## Функционал
- ✅ Фильтрация проектов по технологиям (HTML, CSS, JavaScript, React и др.)
- ✅ Фильтрация по уровню сложности (Начинающий, Средний, Продвинутый)
- ✅ Сохранение состояния фильтров в LocalStorage
- ✅ Сортировка проектов по различным параметрам
- ✅ Адаптивный дизайн для всех устройств
- ✅ Визуальная индикация активных фильтров
- ✅ Сброс фильтров одной кнопкой
- ✅ Подсчет отображаемых проектов

## Структура проекта
- `index.html` - главная страница
- `style.css` - стили проекта
- `script.js` - основная логика приложения
- `data.js` - данные проектов
- `README.md` - документация

## Установка и запуск
1. Скачайте все файлы проекта в одну папку
2. Откройте `index.html` в браузере
3. Проект работает без дополнительной установки

## Используемые технологии
- HTML5 (семантическая верстка)
- CSS3 (Flexbox, Grid, переменные, адаптивность)
- Vanilla JavaScript (ES6+)
- LocalStorage API
- Font Awesome иконки
- Google Fonts (Inter)
